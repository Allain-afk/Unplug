
  # Minimalist Advocacy Page



  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploying to Vercel (manual)

  This project is prepared for static deployment on Vercel.

  Recommended Vercel settings:

  - Framework Preset: `Vite`
  - Build Command: `npm run build`
  - Output Directory: `dist`

  Notes:

  - SPA route fallback is configured in `vercel.json`.
  - Vercel build/cache output is ignored in `.gitignore`.
